# Resumatch - AI ATS Resume Analyzer

AI-powered ATS Resume Analyzer that evaluates resumes against job descriptions and provides structured feedback to help users improve their resumes and increase their chances of passing Applicant Tracking Systems (ATS).

## 🚀 Features

* AI-powered ATS resume scoring
* Upload and analyze PDF resumes
* Extract resume text and key information
* Skill, keyword, and experience analysis
* Compare resumes against job descriptions
* Personalized improvement suggestions
* Real-time resume evaluation
* Clean and responsive user interface

## 🛠️ Tech Stack

* React.js
* TypeScript
* React Router
* Zustand (State Management)
* AI APIs
* PDF Processing Libraries

## 📌 Project Overview

Resumatch is an intelligent resume screening tool that simulates how modern Applicant Tracking Systems (ATS) evaluate resumes.

Users can upload their resumes, provide a job description, and receive an ATS score along with actionable feedback to improve their resume quality and job readiness.

## ⚙️ How It Works

1. Upload a PDF resume.
2. Enter a target job description.
3. AI analyzes skills, keywords, and experience.
4. The application compares resume content with job requirements.
5. An ATS score and detailed feedback are generated.

## 📂 Installation & Setup

Clone the repository:

```bash
git clone https://github.com/your-username/resumatch.git
```

Navigate to the project folder:

```bash
cd resumatch
```

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```
